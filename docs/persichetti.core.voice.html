persichetti.core.voice module

class persichetti.core.voice.Voice(name: str, mode: str, mif_handle: str = 'default')[source]

Bases: object

Represents a single voice in the score, capable of being monophonic or polyphonic. Each voice is associated with a specific Musical Inertial Frame (MIF) for timing.

add_event(motif)[source]

Add a melodic or harmonic motif to the voice’s event list.

Parameters:

motif – The motif to be added (either MelodicMotif or HarmonicMotif).

assign_mif(mif_handle: str, mif_registry: dict)[source]

Assign a Musical Inertial Frame (MIF) to this voice based on its mif_handle. If the MIF doesn’t exist, create a new one.

Parameters:

  • mif_handle – The handle to identify the MIF (e.g., ‘default’).

  • mif_registry – The registry of all MIFs in the score, mapped by their handle.

Parameters:

motif – The motif to be removed.

instrument – The instrument to be assigned to this voice.
